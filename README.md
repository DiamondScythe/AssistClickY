# AssistClickY

Application to help users with various built-in tools like:

•	clipboard monitor
•	clipboard history
•	pasting from clipboard history
•	screenshot
•	audio recording
•	shortcut tray
•	firing custom hotkeys from popup menu
•	more coming soon™

An alpha release version will be released soon.
